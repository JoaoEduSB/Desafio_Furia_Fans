# Desafio_Furia_Fans
Projeto de um protótipo de uma plataforma para fãs da FURIA. Foi desenvolvido essa solução proposta para o desafio Hard da Furia.

## Funcionalidades
- Telas de Registro
- Tela de Login
- Tela principal
- Missões
- Jogos
- Perfil
- Configurações
